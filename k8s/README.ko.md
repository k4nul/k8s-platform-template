# Kubernetes 매니페스트

[English](README.md) | 한국어

이 디렉터리에는 배포 단계와 성격에 따라 나눈 재사용 가능한 Kubernetes 매니페스트가 들어 있습니다.

이름 규칙은 다음과 같습니다.

- `100_*`: 기반 인프라
- `200_*`: 스토리지
- `300_*`: 공용 서비스
- `400_*`, `401_*`: 애플리케이션 예제

아래 명령을 사용하면 프로필이나 애플리케이션 선택에 따라 어떤 디렉터리가 포함되는지 번들 렌더링 전에 확인할 수 있습니다.

```powershell
.\scripts\show-platform-plan.ps1 -Format markdown
```
